##  Project Overview
This project implements a machine learning classifier that categorizes text documents into:

- Sport
- Politics

Three different machine learning models were compared using TF-IDF and n-gram features.

---

##  Dataset
The dataset consists of news articles collected from public sources i.e BBC News Dataset which is already categorized with categories including: sport, politics, business, tech, entertainment
We kept only Classes:
- Sport
- Politics

The data is divided into:

- Training Set (80%)
- Test Set (20%)

---

##  Feature Representation
- Bag of Words
- TF-IDF
- n-grams (Unigram + Bigram)
